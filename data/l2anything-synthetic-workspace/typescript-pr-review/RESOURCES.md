# TypeScript PR Review Resources

## Knowledge

- [TypeScript Handbook: Everyday Types](https://www.typescriptlang.org/docs/handbook/2/everyday-types.html)
  Use for: baseline type annotation vocabulary.
- [TypeScript Handbook: Narrowing](https://www.typescriptlang.org/docs/handbook/2/narrowing.html)
  Use for: review questions around runtime checks and control flow.
- [TypeScript ESLint Rules](https://typescript-eslint.io/rules/)
  Use for: lint-backed evidence when a review comment should become an automated check.

## Wisdom (Communities)

- [Effective TypeScript](https://effectivetypescript.com/)
  Use for: practical explanations of type-system tradeoffs.
