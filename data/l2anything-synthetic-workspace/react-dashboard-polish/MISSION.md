# Mission: React Dashboard Polish

## Why

Turn rough operational screens into quiet, trustworthy dashboards that can be scanned every morning without visual noise.

## Success looks like

- Choose the right density for cards, tables, and sidebars.
- Keep one clear primary action per view.
- Spot when decorative styling is hurting task clarity.

## Constraints

- Product UI, not landing-page UI.
- Prefer semantic tokens and existing components.
- Mobile should be usable but desktop is the main work surface.

## Out of scope

- Brand illustration systems.
- Animation-heavy marketing pages.
